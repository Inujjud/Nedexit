01 — Medical
🏥
أنظمة الحجز الطبي

  • ثلاث مستويات: مريض / طبيب / مدير
  • إدارة الجداول تلقائياً
  • قاعدة بيانات حية — Supabase

🤖

  • Voiceflow + Claude AI
  • واتساب Business API
  • Handoff تلقائي للمختص
Voiceflow + WhatsApp API
03 — Operations
⚙️

  • متعدد الأقسام والمستخدمين
  • وضع فاتح وداكن
HTML + Supabase + Real-time
